We tried to make Termmy as simple as possible, but as everyone is the beginning is difficult, especially when it comes to something new and unknown, we want to give you some advice how to go.

After the first registration, please fill in your basic information, so that we know who you are. After filling in the basic information, you will enter in the application and you can start using it. This will only take you a minute, we promise. üòä

Termmy has an option that can make it easier for you to set up the application. By clicking to the icon (screenshot icon layout is below), you will get options for settings.

By clicking on each of the options, a new page will open, where you can add information (if any in the matter of adding a client, service and random) or to set one of the options (if any let's say in terms of SMS reminder options, online booking pages or similarly).

After each completed setting of an option, that option will turn green, a the percentage that is on initially 0% will increase.
